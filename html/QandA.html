Content Questions (group 3)

Provide a definition for the following programming concepts.

[Authentication and web security]
Authentication is the process of verifying a user or device before allowing access to a system or resources.
[Hypertext transfer protocol (HTTP)]
HTTP is the cornerstone of the world wide web and is used to locate web pages through hypertext links.
[Session management]
Exchange mechanism that handles requests and responses between users and websites.
[Stateless programming]
The program does not maintain a state but instead, passed data would drive the functionalities of the program.

Content Questions (group 4)

Provide a definition for the following types of technologies.

The three major programming control structures.
[Sequence]
Linear execution of code in the order they are written
[Selection]
Decision making through conditional statements like if, if-else and switch
[Iteration]
Repetition of code through loops like for, while. do-while
[Hypertext markup language (HTML) and markup languages]
HTML is the standard markup language for creating Web pages
[Cascading style sheets (CSS)]
Cascading Style Sheets (CSS) is a stylesheet language used to describe the presentation of a document written in HTML or XML (including XML dialects such as SVG, MathML or XHTML). CSS describes how elements should be rendered on screen, on paper, in speech, or on other media.
[Syntax and uses of programming languages]
Syntax is the set of rules that dictate how a program is written and how it should be structured. It includes the appropriate use of keywords, data structures, functions, and loops.
